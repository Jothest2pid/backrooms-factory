THE BACKROOMS

a non-euclidean top-down walking sim

  • WASD / arrows — walk   Shift — run
  • right-click — mine / disassemble / strip carpet / deconstruct
  • left-click — inspect / use machine   C — crafting
  • 1–9 — hotbar   left-click — use held (place / shoot / throw / eat)
  • C — craft/tinker   I — inventory & equip   V — blink
  • B — build   R — rotate   ~ — god   P — debug   L — export log

Rooms are stitched together by doorways — not by space. Walk a loop and the geometry won't add up. Connected rooms hang around you at half opacity; the one you're standing in snaps to full focus.

EQUIPMENT — click a slot to unequip
INVENTORY — click a trinket to equip · I to close
CRAFTING — C to close
